# World Compendium Packs — Gaia-APS-v10

All files are in `packs/`. Same NeDB format as `data/` — use Grep, don't read whole files.

| File | Type | Contains |
|------|------|----------|
| `mentalismo.db` | Item | Mentalism powers available in this campaign |
| `bestiario.db` | Actor | Creature stat blocks (bestiary) |
| `escenas.db` | Scene | Archived/prepared scenes |
| `artefactos-magicos.db` | Item | Magic artifacts and enchanted items |
| `invocaciones-animabf-guote.db` | Item | Summoning/invocation items |
| `npc.db` | Actor | Named NPCs used in the campaign |
| `playlists.db` | Playlist | Music and ambient sound playlists |
| `baileywiki-maps.actors.db` | Actor | Actors from BaileyWiki Maps module (third-party content) |
